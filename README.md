
# Tesla Stock Performance Dashboard

### 🔗 Live Dashboard  
https://public.tableau.com/app/profile/raghad.abdullah4951/viz/TeslaStockPerformanceDashboard/Dashboard3?publish=yes

Dashboard analyzing **Tesla stock price, volatility, and trading volume** using historical data from Kaggle.  
Designed for non‑technical users to quickly understand key market trends.

---

## 📊 Data Source
- Kaggle — Tesla Historical Stock Data  
- Link: https://www.kaggle.com/datasets/varpit94/tesla-stock-data-updated-till-28jun2021?resource=download
- Fields: Date, Open, High, Low, Close, Volume

---

## 🛠 Tools Used
- **Tableau Public** — Interactive dashboard  
- **Figma** — UI layout & design  
- **CSV Dataset** — Kaggle

---

## 📈 Key Features
- KPIs: Current Price, Daily % Change, 52‑Week High/Low, Avg Volume  
- Stock Price Over Time  
- Daily Returns %  
- Volume Over Time  
- Moving Averages  
- Date & Quarter filters

Screenshots included in `/screenshot`.

---

## 🔍 Key Insights
- Tesla shows strong long‑term growth with periods of high volatility.  
- Daily returns show frequent positive/negative swings → high‑risk profile.  
- Volume spikes align with major events and earnings periods.  
- Moving averages highlight upward momentum over time.

---

## ⚠ Assumptions & Limitations
- Historical price data only; no external economic factors included.  
- Dataset may not reflect adjusted prices.  
- Dashboard is descriptive, not predictive.
